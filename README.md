# Script From Airdrop Sambil Rebahan
gunakan dengan bijak 

***git clone script**
git clone
**Create Screen**
```
screen -S teaauto
```
**INSTALL AUTO**
```
chmod +x setup-teatransfer.sh
./setup-teatransfer.sh
```
**Isi Private key dan alamat token (token tele boleh abaikan)**
```
nano .env
```
**mainkan**
```
node teatransfer.js
```
**cara masuk screen**
```
screen -r teaauto
```
